# 📱 TaskManager - Aplicación Móvil de Gestión de Tareas

## 📋 Descripción del Proyecto

**TaskManager** es una aplicación móvil diseñada para ayudar a los usuarios a organizar y gestionar sus tareas diarias de manera eficiente. La aplicación permite crear, editar, eliminar y visualizar tareas, con un enfoque en la simplicidad y la productividad.

**Temática:** Gestión de tareas y productividad personal.

## 🎯 Definición del Problema

En la actualidad, las personas enfrentan dificultades para organizar sus actividades diarias, lo que genera estrés, olvidos y pérdida de productividad. Las herramientas existentes suelen ser complejas o están llenas de funciones innecesarias que distraen al usuario.

**TaskManager** resuelve este problema ofreciendo:

- Una interfaz simple e intuitiva
- Gestión básica de tareas (CRUD)
- Organización por categorías
- Recordatorios visuales
- Acceso rápido desde el móvil
